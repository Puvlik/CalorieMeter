# CalorieMeter
**Стек: iOS 18.0, Xcode 16, Swift 6 + SwiftUI, CoreData**

Данное приложение поможет вам подсчитать калории, которые вы употребляете в пищу в тот или иной день. 
Съели вкусняшку? Добавьте её с калориями. Допустили помарку или ошибку при добавлении? Исправьте. Съели слишком много и совесть давит? Удалите — никто не узнает! 
Приложение подсчитает все калории и честно расскажет, сколько вы «съели»!
